---
name: tasks2md
description: Use this prompt to convert all tasks to markdown format. It will execute a script that converts all tasks to markdown format and saves them in the .taskmaster/tasks directory.
---

Execute one of the follow commands to convert all tasks to markdown format and save them in the .taskmaster/tasks directory.
```shell
python3 .github/prompts/scripts/convert_tasks.py --to_markdown
python3 $HOME/.gemini/commands/scripts/convert_tasks.py --to_markdown
```
