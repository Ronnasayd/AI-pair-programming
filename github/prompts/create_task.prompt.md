use the tool to create the task:

- readFile {file_path} -> reads the content of a file to gather necessary information for creating the task
- my_mcp_task_create {task_description} -> provides instructions for the MCP developer to create the task
- sequentialthinking -> provides a step-by-step plan to create the task
- codebase_investigator -> investigates the codebase to gather necessary information for creating the task
